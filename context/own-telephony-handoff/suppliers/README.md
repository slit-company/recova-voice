# suppliers

공급자 통화/메일 결과를 저장하는 폴더다.

작성 방법:

1. `../05-supplier-intake-template.md`를 복사한다.
2. 파일명은 `YYYYMMDD-업체명.md`로 쓴다.
3. credential/password/API key 원문은 저장하지 않는다.
4. 공급자 PDF/견적서 원본은 repo에 넣지 말고, 보안 저장 위치만 메모한다.

예:

```text
20260710-onnuri070.md
20260710-wyz070-ip-link.md
20260710-kt-enterprise.md
```
